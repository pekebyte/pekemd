Markdown block for Wordpress Editor

== Description ==

 It allows users to write content using Markdown syntax, which is then automatically converted into html within the Gutenberg interface. This plugin is ideal for those who prefer the simplicity and speed of Markdown but want to take advantage of Gutenberg's advanced layout options. Enjoy a smoother writing experience with live previews, easy formatting, and enhanced productivity.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/pekemd` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
